# una-ihcux-pratica01

Durante a aula de Interação Humano-Computador e UX do dia 19 de março de 2026, a turma fez uma atividade prática de 3 exercícios envolvendo terminal.

No primeiro exercício chamado "O Detetive de Arquivos", utilizei o comando dir para navegar até a minha pasta de Downloads e criei a pasta Joazinho utilizando o comando mkdir, depois utilizei os comandos de cd (.) para voltar e dir para acessar a pasta de Documents.

No segundo exercício chamado "Hierarquia de Pastas", criei a seguinte estrutura usando apenas o comando mkdir e cd: Projetos > Exercicios > Logica. Depois, voltei para a pasta Projetos usando apenas um comando (cd ../..).

No terceiro e último exercício chamado "Versão dos Softwares", utilizei os comandos "dotnet --version - git --version - node -v" para encontrar determinados programas no computador e descobrir suas versões.
